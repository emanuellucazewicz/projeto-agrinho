# 🌾 Portal Agrinho 2026 - Estatísticas do Agronegócio

Este é um projeto web desenvolvido para o **Programa Agrinho**, com o objetivo de apresentar dados estatísticos reais, históricos e confiáveis sobre o desenvolvimento do agronegócio brasileiro. O portal permite que o usuário filtre informações por anos específicos, destacando a evolução da produção de grãos e marcos importantes do setor.

---

## 🎨 Identidade Visual e Cores

A paleta de cores do site foi planejada para refletir o tema do campo e a identidade tradicional do Programa Agrinho, utilizando tons que remetem à natureza, à tecnologia agrícola e à terra:

* **Verde Escuro (`#1b4d3e`)**: Representa a força do agronegócio e a sustentabilidade. Usado em cabeçalhos e rodapés.
* **Verde Folha (`#2e7d32`)**: Cor de destaque para botões, cards e elementos importantes.
* **Verde Claro (`#e8f5e9`)**: Fundo suave para garantir uma leitura confortável e realçar tabelas.
* **Marrom Terra (`#795548`)**: Conexão com o solo e a agricultura, utilizado para detalhes e bordas.

---

## 🔒 Compromisso com a Verdade (Combate a Fake News)

Um dos pilares deste projeto é a **credibilidade**. Nenhuma informação exibida no site é fictícia. O sistema de filtragem por anos utiliza dados oficiais coletados diretamente de fontes governamentais e institutos de pesquisa de grande prestígio:

* **IBGE** (Instituto Brasileiro de Geografia e Estatística) - Dados do LSPA e PAM.
* **CONAB** (Companhia Nacional de Abastecimento) - Dados de safras e produtividade.
* **MAPA** (Ministério da Agricultura e Pecuária).

---

## 🚀 Funcionalidades do Código

* **Layout Responsivo**: O site se adapta automaticamente a telas de computadores, tablets e celulares (CSS Grid e Flexbox).
* **Linha do Tempo Dinâmica**: Sistema em **JavaScript Puro (Vanilla JS)** que altera os dados na tela em tempo real assim que o usuário seleciona o ano no menu, sem a necessidade de recarregar a página.
* **Organização Limpa**: Exibição de dados através de cards de destaque e tabelas estruturadas.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização personalizada e design responsivo com variáveis nativas.
* **JavaScript**: Lógica de manipulação do DOM para atualização dinâmica dos dados.
